## 📦 Helpers

### Gerador de Classes NFSE

> **Autor:** [RenatoMAP77](https://github.com/RenatoMAP77)

Age em conjunto com um agente CLI para geração automática de classes NFSE, utilizando o template do projeto [nfse-builder](https://github.com/RenatoMAP77/nfse-builder).

🔗 **Repositório:** [nfse-builder](https://github.com/RenatoMAP77/nfse-builder)

---

### Gerador de Telas Fiori

> **Autores:** [Lealwbs](https://github.com/Lealwbs) e [RenatoMAP77](https://github.com/RenatoMAP77)

Automatiza a instalação do Fiori no ambiente do cliente, facilitando todo o processo de setup.

🔗 **Repositório:** [fiori-automator](https://github.com/Lealwbs/fiori-automator)

---

### ZipGithubDownloader

> **Autor:** [RenatoMAP77](https://github.com/RenatoMAP77)

Interface gráfica intuitiva para baixar código-fonte (.zip) de múltiplos repositórios do GitHub baseado em tags específicas. Visualize todas as tags disponíveis de cada repositório e escolha exatamente qual versão deseja baixar.

🔗 **Repositório:** [ZipGithubDownloader](https://github.com/RenatoMAP77/ZipGithubDowloader)
